## Featured Projects

### LogikaQBits – Token Factory dApp | Solana Token-2022
[![Solana](https://img.shields.io/badge/Solana-Token--2022-9945FF?logo=solana)](https://github.com/AnelHenning2-collab/LogikaQBits-dApp)
[![Deep Learning](https://img.shields.io/badge/Deep%20Learning-Inference%20Augmented-76B900?logo=nvidia)](https://github.com/AnelHenning2-collab/LogikaQBits-dApp)

A full-stack decentralized application (dApp) built on Solana using the Token-2022 program, designed as a **token factory** — a production-grade pipeline for minting, augmenting, and deploying on-chain tokens at scale.

This project is directly aligned with the themes from **NVIDIA GTC 2026**, where Jensen Huang framed the token as the fundamental unit of modern AI, and announced NVIDIA Dynamo — an inference operating system that boosts token throughput 7x to drive revenue. LogikaQBits applies the same factory model and scale-up/scale-out architecture principles to blockchain token infrastructure.

**[View Repository →](https://github.com/AnelHenning2-collab/LogikaQBits-dApp)**
**[Live Demo →](https://anelhenning2-collab.github.io/LogikaQBits-dApp/)**

**Skills demonstrated:**
- Token factory design — minting pipeline with throughput and cost optimization
- Scale up (richer token metadata, Token-2022 extensions) and scale out (multi-wallet ATA deployment)
- Deep learning token analogy — inference augmentation, KV cache, MoE routing mapped to blockchain
- Ethereum Solidity smart contract with escrow and Checks-Effects-Interactions pattern
- Authority revocation for immutable, trustless supply (analogous to frozen model weights)
- Responsive multi-step Web3 UX with wallet integration and error handling
- Unit testing with Jest and @solana/spl-token
