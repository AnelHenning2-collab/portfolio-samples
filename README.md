### LogikaQBits – Solana Token-2022 dApp
[![Solana](https://img.shields.io/badge/Solana-Token--2022-9945FF?logo=solana)](https://github.com/AnelHenning2-collab/LogikaQBits-dApp)
[![Ethereum](https://img.shields.io/badge/Ethereum-Solidity-3C3C3D?logo=ethereum)](https://github.com/AnelHenning2-collab/LogikaQBits-dApp)

A full-stack decentralized application built on Solana using the Token-2022 program. Demonstrates end-to-end dApp development including on-chain metadata, authority revocation, responsive GUI, and cross-chain smart contract development.

**[View Repository →](https://github.com/AnelHenning2-collab/LogikaQBits-dApp)**

**Skills demonstrated:**
- Solana Token-2022 mint initialization and metadata assignment
- Authority revocation for trustless, supply-capped tokens
- Ethereum Solidity smart contract with escrow pattern
- Responsive multi-step Web3 UI/UX design
- Unit testing with Jest (@solana/spl-token)
- Cross-chain development (Solana + Ethereum)
